A no-frills, twitter updater(title and url) when post/page is published. Can be used as a standalone plugin or integrated with the URL Shortener Plugin (if installed). 

== Description ==


*NOTE: This plugin does not have oauth support. Until I have time to further develop this plugin, it will not work with the new twitter... *


[Simply Tweeted](http://fusedthought.com/downloads/tweeted-wordpress-plugin/ "Tweeted") is a simple, no-frills plugin which updates twitter with the post title and post url upon post or page publishing. 

Simply Tweeted is developed as a standalone, but it also provides integration with the [URL Shortener](http://fusedthought.com/downloads/url-shortener-wordpress-plugin/ "URL Shortener") Plugin (Ver 2.1 and above).


**Support via:**

== Installation ==

1. Upload files to your `/wp-content/plugins/` directory (preserve sub-directory structure if applicable)
1. Activate the plugin through the 'Plugins' menu in WordPress
